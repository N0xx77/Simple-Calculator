# Simple Calculator

A lightweight calculator built with HTML, CSS, and vanilla JavaScript. It supports basic arithmetic operations, decimal input, delete, and clear actions.

## Features

- Basic operations: addition, subtraction, multiplication, and division
- Decimal support
- Delete last character with `Del`
- Clear the display with `C`
- Responsive layout for smaller screens

## Project Structure

- `index.html` - Main entry point for the calculator
- `calculator.html` - Original HTML file for the calculator UI
- `calculator.css` - Styles for the calculator layout and buttons
- `calculator.js` - Calculator interaction and expression handling

## How to Run

1. Clone or download the repository.
2. Open `index.html` in a browser.
3. Use the calculator buttons to build an expression.
4. Press `=` to evaluate the result.

## Controls

- `0-9` - Number input
- `+`, `-`, `x`, `/` - Arithmetic operators
- `.` - Decimal point
- `Del` - Remove the last character
- `C` - Clear the display
- `=` - Compute the expression

## Notes

- The calculator uses the browser's JavaScript engine to evaluate expressions.
- Division by zero or malformed expressions will display `Error`.
